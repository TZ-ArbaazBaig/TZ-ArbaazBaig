### 🤖 InterviewPrep AI
> `react` `node.js` `mongodb` `pinecone` `groq api` `clerk` `express.js`

- Full-stack RAG platform — built entire pipeline from ingestion to automated scoring
- Vector search with **Pinecone** + LLM evaluation via **Groq API**
- End-to-end authentication and role-based access via **Clerk**
- Signal detection logic analogous to fraud risk flagging
- [Live Demo →](https://interview-prep-nine-zeta.vercel.app)

### 🔍 Credit Card Fraud Detection
> `python` `pandas` `scikit-learn` `smote` `random-forest` `matplotlib` `seaborn`

- Analysed **284,807 real-world transactions** — identified fraud patterns using EDA
- Tackled severe class imbalance (**0.17% fraud rate**) with SMOTE oversampling
- Random Forest → **AUC-ROC: 0.9688** · Fraud Precision: **0.82** · F1: **0.82**
- Visualised risk signals — correlation heatmaps, ROC curves, feature importance charts
- [GitHub →](https://github.com/TZ-ArbaazBaig/credit-card-fraud-detection)
