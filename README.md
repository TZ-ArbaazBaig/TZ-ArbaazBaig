### 🔍 credit card fraud detection
> `python` `pandas` `scikit-learn` `smote` `random-forest` `matplotlib` `seaborn`

- Analysed **284,807 transactions** — identified fraud patterns using EDA
- Tackled **0.17% fraud rate** with SMOTE oversampling — improved minority class recall
- Random Forest → **AUC-ROC: 0.9688** · Fraud Precision: **0.82** · F1: **0.82**
- Visualised risk signals — correlation heatmaps, ROC curves, feature importance
